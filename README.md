# stoat-and-ferret

[![License: AGPL v3](https://img.shields.io/badge/License-AGPL_v3-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)

[Alpha] AI-driven video editor with hybrid Python/Rust architecture - not production ready.

**v089 (R3 Wave 3a+4):** ZoompanBuilder, CurvesBuilder, VignetteBuilder, and HueRotationBuilder (FFmpeg filter builders with KneeString per-channel colour grading and single-quote expression wrapping); procedural shape generators (Spiral, RadialBurst, Checkerboard, ConcentricRings) in a new Rust `shapes/` module using the `image` crate; 8 new Wave 3a smoke tests; Timeline-T window verification (PRs #671–#679). See [CHANGELOG.md](docs/CHANGELOG.md) for release details.

# Impulse Response Assets

Synthetic impulse response files bundled with stoat-ferret for convolution reverb.

## Format

- Sample rate: 48 kHz
- Channels: stereo (2)
- Bit depth: 16-bit PCM
- Encoding: exponential-decay noise burst

## Presets

| File | Decay | Duration |
|------|-------|----------|
| `hall_small.wav` | ~3 s⁻¹ | 1.5 s |
| `room_medium.wav` | ~6 s⁻¹ | 0.8 s |
| `plate.wav` | ~2 s⁻¹ | 2.0 s |

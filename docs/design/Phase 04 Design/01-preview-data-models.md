# Phase 4: Preview Data Models

## Overview

Phase 4 introduces proxy file management, HLS-based preview sessions, playback state tracking, and visual aids (waveforms, thumbnail strips). These data models extend the existing Video, Project, and Job schemas.

## Proxy File Model

### ProxyFile (Python - Pydantic)

```python
class ProxyQuality(str, Enum):
    LOW = "540p"      # for 1080p sources
    MEDIUM = "720p"   # for 4K sources
    HIGH = "1080p"    # passthrough for <=1080p sources

class ProxyStatus(str, Enum):
    PENDING = "pending"
    GENERATING = "generating"
    READY = "ready"
    FAILED = "failed"
    STALE = "stale"     # source modified after proxy generated

class ProxyFile(BaseModel):
    id: str
    source_video_id: int
    quality: ProxyQuality
    file_path: str           # path to proxy file
    file_size_bytes: int
    status: ProxyStatus
    generated_at: str | None  # ISO 8601
    source_checksum: str      # detect stale proxies
    last_accessed_at: str | None  # ISO 8601, updated on proxy read (for LRU eviction)
```

### Proxy Quality Selection Logic

```python
def select_proxy_quality(source_width: int, source_height: int) -> ProxyQuality:
    """Automatic proxy quality based on source resolution."""
    pixels = source_width * source_height
    if pixels > 3840 * 2160:     # >4K
        return ProxyQuality.MEDIUM  # 720p
    elif pixels > 1920 * 1080:   # >1080p
        return ProxyQuality.MEDIUM  # 720p
    elif pixels > 1280 * 720:    # >720p
        return ProxyQuality.LOW     # 540p
    else:
        return ProxyQuality.HIGH    # passthrough, no proxy needed
```

## Preview Session Model

### PreviewSession (Python - Pydantic)

```python
class PreviewStatus(str, Enum):
    INITIALIZING = "initializing"
    GENERATING = "generating"     # FFmpeg producing HLS segments
    READY = "ready"               # manifest available for playback
    SEEKING = "seeking"           # regenerating from seek point
    ERROR = "error"
    EXPIRED = "expired"           # session timed out

class PreviewSession(BaseModel):
    id: str                       # session UUID
    project_id: str
    status: PreviewStatus
    quality: ProxyQuality
    manifest_url: str | None      # HLS .m3u8 URL when ready
    duration: float               # timeline duration in seconds
    current_position: float       # last known playback position
    created_at: str               # ISO 8601
    expires_at: str               # session expiry (configurable TTL)
    segment_duration: float       # HLS segment length (default 2.0s, ge=1.0, le=6.0)
    segments_generated: int       # progress tracking
    segments_total: int           # estimated total segments
```

### HLS Manifest and Segments

Preview sessions produce HLS output in a managed directory:

```
data/previews/{session_id}/
  manifest.m3u8        # HLS master playlist
  segment_000.ts       # MPEG-TS segments
  segment_001.ts
  ...
```

The manifest is a standard HLS VOD playlist generated by FFmpeg:

```
#EXTM3U
#EXT-X-VERSION:3
#EXT-X-TARGETDURATION:4
#EXT-X-MEDIA-SEQUENCE:0
#EXTINF:4.000,
segment_000.ts
#EXTINF:4.000,
segment_001.ts
...
#EXT-X-ENDLIST
```

## Playback State Model

### PlaybackState (Python - Pydantic, also used in frontend store)

```python
class PlaybackMode(str, Enum):
    STOPPED = "stopped"
    PLAYING = "playing"
    PAUSED = "paused"
    SEEKING = "seeking"
    BUFFERING = "buffering"
    LOADING = "loading"

class LoopMode(str, Enum):
    NONE = "none"
    TIMELINE = "timeline"     # loop entire timeline
    SELECTION = "selection"   # loop selected range

class PlaybackState(BaseModel):
    mode: PlaybackMode
    position: float           # current time in seconds
    duration: float           # total duration
    volume: float             # 0.0 to 1.0
    muted: bool
    playback_rate: float      # 0.5x to 2.0x
    loop_mode: LoopMode
    loop_start: float | None  # for selection loop
    loop_end: float | None
    buffered_to: float        # how far ahead is buffered
    quality: ProxyQuality
    latency_ms: float         # seek-to-frame latency
```

### Playback State Machine

```
                    ┌─────────┐
                    │ STOPPED │
                    └────┬────┘
                         │ start_preview
                    ┌────▼────┐
                    │ LOADING │

              ┌──────────▼──────────┐
```

## Thumbnail Strip Model

### ThumbnailStrip (Python - Pydantic)

```python
class ThumbnailStrip(BaseModel):
    video_id: int
    strip_path: str          # path to generated strip image
    frame_count: int         # number of frames in strip
    frame_width: int         # width of each frame (pixels)
    frame_height: int        # height of each frame (pixels)
    interval_seconds: float  # time between frames
    total_duration: float    # video duration covered
    generated_at: str        # ISO 8601
```

Thumbnail strips are horizontal image strips (sprite sheets) used for seek preview tooltips. FFmpeg generates them as a single image with frames side-by-side:

```
┌──────┬──────┬──────┬──────┬──────┬──────┐
│ 0:00 │ 0:05 │ 0:10 │ 0:15 │ 0:20 │ 0:25 │
└──────┴──────┴──────┴──────┴──────┴──────┘
```

Default: 160x90px frames at 5-second intervals.

## Waveform Model

### Waveform (Python - Pydantic)

```python
class WaveformFormat(str, Enum):
    PNG = "png"       # rendered waveform image
    JSON = "json"     # raw amplitude data for frontend rendering

class Waveform(BaseModel):
    video_id: int
    format: WaveformFormat
    file_path: str           # path to waveform file
    samples_per_second: int  # amplitude sample rate
    duration: float          # audio duration
    channels: int            # mono=1, stereo=2
    generated_at: str        # ISO 8601
```

Waveforms are generated by FFmpeg's `showwavespic` filter (PNG) or `astats` filter (JSON data). Used for audio track visualization in the timeline.

## Preview Cache Model

### PreviewCache (Python - internal)

```python
class CacheEntry(BaseModel):
    session_id: str
    project_id: str
    created_at: str
    last_accessed: str
    size_bytes: int
    segment_count: int

class PreviewCacheConfig(BaseModel):
    max_size_bytes: int = 1_073_741_824   # 1 GB default
    max_sessions: int = 5                  # max concurrent preview sessions
    session_ttl_seconds: int = 3600        # 1 hour default
    eviction_policy: str = "lru"           # least recently used
```

Cache eviction runs on a periodic background task and on new session creation when limits are exceeded.

## Database Schema Additions

```sql
-- Proxy files table
CREATE TABLE IF NOT EXISTS proxy_files (
    id TEXT PRIMARY KEY,
    source_video_id INTEGER NOT NULL REFERENCES videos(id),
    quality TEXT NOT NULL,
    file_path TEXT NOT NULL,
    file_size_bytes INTEGER NOT NULL DEFAULT 0,
    status TEXT NOT NULL DEFAULT 'pending',
    source_checksum TEXT NOT NULL,
    generated_at TEXT,
    last_accessed_at TEXT,
    created_at TEXT NOT NULL DEFAULT (datetime('now')),
    UNIQUE(source_video_id, quality)
);

-- Preview sessions table
CREATE TABLE IF NOT EXISTS preview_sessions (
    id TEXT PRIMARY KEY,
    project_id TEXT NOT NULL REFERENCES projects(id),
    status TEXT NOT NULL DEFAULT 'initializing',
    quality TEXT NOT NULL,
    manifest_path TEXT,
    duration REAL NOT NULL DEFAULT 0.0,
    segment_duration REAL NOT NULL DEFAULT 2.0,
    segments_generated INTEGER NOT NULL DEFAULT 0,
    segments_total INTEGER NOT NULL DEFAULT 0,
    created_at TEXT NOT NULL DEFAULT (datetime('now')),
    expires_at TEXT NOT NULL
);

-- Thumbnail strips table
CREATE TABLE IF NOT EXISTS thumbnail_strips (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    video_id INTEGER NOT NULL REFERENCES videos(id),
    strip_path TEXT NOT NULL,
    frame_count INTEGER NOT NULL,
    frame_width INTEGER NOT NULL DEFAULT 160,
    frame_height INTEGER NOT NULL DEFAULT 90,
    interval_seconds REAL NOT NULL DEFAULT 5.0,
    generated_at TEXT NOT NULL DEFAULT (datetime('now')),
    UNIQUE(video_id)
);

-- Waveforms table
CREATE TABLE IF NOT EXISTS waveforms (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    video_id INTEGER NOT NULL REFERENCES videos(id),
    format TEXT NOT NULL DEFAULT 'png',
    file_path TEXT NOT NULL,
    samples_per_second INTEGER NOT NULL DEFAULT 10,
    duration REAL NOT NULL,
    channels INTEGER NOT NULL DEFAULT 2,
    generated_at TEXT NOT NULL DEFAULT (datetime('now')),
    UNIQUE(video_id, format)
);
```
